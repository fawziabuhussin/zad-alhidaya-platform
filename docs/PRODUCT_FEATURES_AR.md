## نظرة عامة على المنصة

**زاد الهداية** منصة تعليمية إسلامية متكاملة، صُمِّمت لتقديم الدورات الشرعية بأسلوب منظم وميسّر للطلاب في مختلف أنحاء العالم. تتميز المنصة بواجهة عربية أصيلة وتصميم مستوحى من الهوية الإسلامية.

تدعم المنصة ثلاثة أنواع من المستخدمين، لكلٍّ منهم صلاحيات وإمكانيات محددة:
- **الطلاب**: المتعلمون الذين يتابعون الدورات ويؤدون التقييمات
- **المدرسون**: المعلمون الذين يُعدّون المحتوى ويتابعون تقدم الطلاب
- **المسؤولون**: المشرفون الذين يديرون المنصة بالكامل

## 1. أنواع المستخدمين والصلاحيات

### 1.1 الطالب
الطلاب هم المتعلمون الأساسيون في المنصة. يمكنهم تصفح الدورات، والتسجيل فيها، ومتابعة المحتوى، وإتمام التقييمات، ومتابعة تقدمهم الأكاديمي.

### 1.2 المدرس
المدرسون يُنشئون ويديرون المحتوى التعليمي. يصممون الدورات، ويُعدّون التقييمات، ويُصححون أعمال الطلاب، ويردون على استفساراتهم.

### 1.3 المسؤول (المشرف العام)
المسؤولون لديهم صلاحيات كاملة على المنصة. يديرون المستخدمين، والفئات، والتسجيلات، ولديهم جميع صلاحيات المدرسين على كافة الدورات.

## 3. إدارة الدورات

### 3.1 تنظيم الدورات
| الإمكانية | الوصف |
|-----------|-------|
| الفئات | تنظيم الدورات ضمن فئات لسهولة الوصول |
| قائمة الدورات | كتالوج قابل للبحث يعرض جميع الدورات المنشورة |
| التصفية حسب الفئة | تصفية الدورات حسب الفئة المطلوبة |
| البحث | البحث في الدورات حسب العنوان أو الوصف |

### 3.2 هيكل الدورة
تتكون كل دورة من:
- **تفاصيل الدورة**: العنوان، الوصف، صورة الغلاف، معلومات المدرس
- **الوحدات**: أقسام منظمة داخل الدورة
- **الدروس**: وحدات تعليمية فردية داخل الوحدات
- **المواد والمرفقات**: ملفات وروابط قابلة للتحميل
- **التقييمات**: الامتحانات، الواجبات، والاختبارات القصيرة

### 3.3 إنشاء وتعديل الدورات (للمدرسين والمسؤولين)
| الإمكانية | الوصف |
|-----------|-------|
| الإنشاء اليدوي | إنشاء دورات مع وحدات ودروس مخصصة |
| الاستيراد من YouTube | إنشاء هيكل الدورة تلقائياً من قائمة تشغيل YouTube |
| وضع المسودة | حفظ الدورات كمسودة قبل النشر |
| النشر | نشر الدورات لتصبح متاحة للطلاب |
| صورة الغلاف | رفع صور غلاف مخصصة للدورات |
| إعداد التقييم | ضبط أوزان التقييم (القراءة، الواجبات، الامتحانات) |
| تعديل الدورة | تعديل جميع تفاصيل الدورة والوحدات والدروس |
| إعادة ترتيب المحتوى | تغيير ترتيب الوحدات والدروس داخل الدورة |
| حذف الدورة | حذف الدورة بالكامل مع جميع محتوياتها |

### 3.4 أنواع الدروس
| النوع | الوصف |
|-------|-------|
| دروس الفيديو | فيديوهات YouTube مدمجة مع مشغل داخلي |

## 7. التواصل والدعم

### 7.1 الأسئلة المتعلقة بالدروس (سؤال متعلق بالدرس)
يمكن للطلاب طرح أسئلة حول دروس محددة والحصول على إجابات من المدرسين.

**إمكانيات الطالب:**
| الإمكانية | الوصف |
|-----------|-------|
| طرح سؤال | إرسال أسئلة من أي صفحة درس |
| عرض أسئلتي | مشاهدة جميع الأسئلة المطروحة في صفحة "أسئلتي" |
| حالة السؤال | متابعة الحالة (قيد الانتظار، تمت الإجابة) |
| عرض الإجابات | قراءة ردود المدرس أو المسؤول |
| حذف الأسئلة | حذف الأسئلة التي لم تتم الإجابة عليها بعد |

**إمكانيات المدرس والمسؤول:**
| الإمكانية | الوصف |
|-----------|-------|
| إشعارات الأسئلة | شارة تعرض عدد الأسئلة الجديدة في الوقت الفعلي |
| عرض الأسئلة | مشاهدة أسئلة الطلاب في الدورات المعنية |
| الإجابة على الأسئلة | تقديم إجابات مكتوبة لأسئلة الطلاب |
| حذف الأسئلة | حذف الأسئلة غير المناسبة أو المحلولة |
| التصفية حسب الحالة | عرض الأسئلة حسب الحالة (قيد الانتظار أو تمت الإجابة) |
| التصفية حسب الدورة | عرض أسئلة دورة محددة |

### 7.2 التبليغ عن أخطاء المحتوى (تبليغ عن خطأ بالمادة)
يمكن للمستخدمين التبليغ عن الأخطاء أو المشاكل الموجودة في محتوى الدورات.

**إمكانيات المُبلِّغ:**
| الإمكانية | الوصف |
|-----------|-------|
| التبليغ عن خطأ | إرسال تبليغ من أي صفحة درس |
| أنواع الأخطاء | خطأ في الفيديو، خطأ في النص، خطأ في الصوت، خطأ في المرفقات، محتوى ناقص، أخرى |
| تحديد التوقيت | تحديد الوقت الدقيق لأخطاء الفيديو |
| وصف الخطأ | تقديم وصف تفصيلي للخطأ |
| عرض تبليغاتي | متابعة التبليغات المرسلة وحالتها |
| حالة التبليغ | مشاهدة الحالة (جديد، قيد المراجعة، تم الحل، مرفوض) |

**إمكانيات المراجع (المدرسين والمسؤولين):**
| الإمكانية | الوصف |
|-----------|-------|
| إشعارات التبليغات | شارة تعرض عدد التبليغات الجديدة في الوقت الفعلي |
| عرض التبليغات | مشاهدة جميع التبليغات (للمسؤول) أو تبليغات الدورات المعنية (للمدرس) |
| تحديث الحالة | تغيير حالة التبليغ أثناء المعالجة |
| إضافة ملاحظات المراجعة | تقديم ملاحظات توضح الحل |
| حذف التبليغات | حذف التبليغات المحلولة أو غير الصالحة |
| التصفية حسب الحالة | عرض التبليغات حسب الحالة الحالية |
| التصفية حسب الدورة | عرض تبليغات دورة محددة |

## ملخص الإمكانيات الرئيسية

### للطلاب
1. **تعلّم بوتيرتك** - الوصول لدروس الفيديو والنص في أي وقت
2. **تابع تقدمك** - متابعة بصرية للتقدم عبر الدورات
3. **أثبت معرفتك** - إتمام الامتحانات والواجبات مع ملاحظات فورية
4. **احصل على الدعم** - طرح الأسئلة مباشرة على المدرسين
5. **أبلغ عن المشاكل** - ساهم في تحسين جودة المحتوى

### للمدرسين
1. **إنشاء دورات بسهولة** - بناء الدورات يدوياً أو استيرادها من YouTube
2. **تقييمات مرنة** - إنشاء امتحانات وواجبات واختبارات
3. **رؤى عن الطلاب** - متابعة تقدم الطلاب ودرجاتهم
4. **تواصل مباشر** - الإجابة على أسئلة الطلاب بكفاءة
5. **جودة المحتوى** - استلام ومعالجة تبليغات الأخطاء

### للمسؤولين
1. **إشراف كامل** - رؤية شاملة لجميع نشاطات المنصة
2. **إدارة المستخدمين** - التحكم في صلاحيات الوصول والأدوار
3. **تنظيم المحتوى** - إدارة الفئات والهيكل العام
4. **ضمان الجودة** - مراقبة وحل مشاكل المحتوى
5. **صحة المنصة** - متابعة إحصائيات التسجيل والتفاعل
